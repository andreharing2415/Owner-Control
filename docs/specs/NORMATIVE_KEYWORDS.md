# Mapeamento de Palavras-chave Normativas

## Objetivo
Definir palavras-chave por etapa para apoiar busca web de normas e classificacao normativa.

## Requisitos funcionais
- Gerar palavras-chave por etapa para uso em busca web.
- Versionar o mapeamento com data e responsavel.
- Permitir ajustes com rastreabilidade.

## Dados de entrada
- Etapas padrao (ver docs/DOMAIN_TAXONOMY.md).
- Disciplina tecnica principal.
- Tipo de obra e localizacao (opcional).

## Dados de saida
- Lista versionada de palavras-chave por etapa.

## Versao
- Versao: v1.0
- Data: 2026-02-16
- Responsavel: Product

## Mapeamento por etapa
1. Planejamento e Projeto
Palavras-chave: licenciamento de obra, projeto executivo, memoriais, normas tecnicas, acessibilidade, seguranca do trabalho.

2. Preparacao do Terreno
Palavras-chave: terraplanagem, movimentacao de solo, drenagem provisoria, protecao ambiental, estabilidade de taludes.

3. Fundacoes e Estrutura
Palavras-chave: fundacoes, estacas, concreto armado, armações, ensaios de resistencia, cobrimento, fck.

4. Alvenaria e Cobertura
Palavras-chave: alvenaria estrutural, vedacao, impermeabilizacao, telhado, estanqueidade, juntas.

5. Instalacoes e Acabamentos
Palavras-chave: instalacao eletrica, SPDA, aterramento, instalacao hidraulica, teste de pressao, acabamentos.

6. Entrega e Pos-obra
Palavras-chave: vistoria final, termo de entrega, manual do proprietario, as built, garantia, manutencao preventiva.

## Criterios de aceite
- Palavras-chave definidas para todas as 6 etapas.
- Versao e data registradas.
- Aprovado e versionado pelo Product.
